
## Abstruct 
This is a [Next.js](https://nextjs.org/) project, with [Ant Desingn Mobile 5.0](https://mobile.ant.design/zh) integrated.
## Quick Start

**Initialize the project**
```bash
npm install
#or
yarn 
```
**Start up**

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser .
